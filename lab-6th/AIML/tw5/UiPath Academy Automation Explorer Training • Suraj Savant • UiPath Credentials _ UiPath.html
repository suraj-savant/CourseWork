The UiPath Academy Automation Explorer Training is a collection of curated courses covering the most essential topics in automation development for aspiring automation developers, technical business analysts, technical project managers, and other automation roles. The training covers the main capabilities of UiPath Studio with the modern experience, starting with an overview of the UiPath Business Automation Platform, building an initial process with UiPath Studio, using variables and arguments, enabling dynamic data handling, control flow and automation technologies used in business like Excel and others with user interface automation. Throughout the training, users learn from a mix of videos, hands-on practices, knowledge checks, and reflection spaces. They build automations across multiple use-cases and industries. This curriculum has a final assessment.
